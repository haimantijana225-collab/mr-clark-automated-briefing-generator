# Mr. Clarke's Automated Briefing Generator

An AI-powered system that ingests documents and automatically generates a structured PowerPoint presentation.

## Features

- Document ingestion
- Automated text summarization
- Dynamic slide generation
- Chart creation from numerical data

## Tech Stack

Python  
Flask  
PyPDF2  
Matplotlib  
python-pptx  

## Project Structure

backend/
frontend/
uploads/

## How It Works

1 Upload PDF document
2 System extracts text
3 Content is summarized
4 Slides are generated automatically
5 PowerPoint presentation is downloaded
